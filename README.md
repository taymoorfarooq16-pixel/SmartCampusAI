# Smart Campus AI

A Django-based Smart Campus Management System with:

- Student Dashboard
- Attendance Tracking
- AI Attendance Advisor
- Subject Management
- Event Notifications
- Lost & Found Portal
- AI Chat Assistant

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap
- GitHub Models API
